The Tax Record API

A readonly API for retrieval of national tax records.

Operations:

Add a New Record

Retrieve Records

Add a new Record

Request

POST

/publicitytax/records

{
    "statement_date": "14/07/2013",
    "amount_due": "2399.00"
}

Sample Response

{
    "level": "3",
    "result": "successful"
}

Retrieve Tax Records

Request

GET

/publicitytax/records

Sample Response

{
    "level": "3",
    "records": [
        {
            "statement_date": "14/07/2013",
            "amount_due": "2399.00"
        }
    ]
}
